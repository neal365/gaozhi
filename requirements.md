# FlowState 心流写作 - 需求文档

## 1. 产品概述

### 1.1 产品名称
FlowState（心流写作）

### 1.2 产品定位
基于浏览器的沉浸式写作 Web 应用，核心特色为仿真稿纸背景与手写书法字体渲染，帮助创作者进入深度专注的写作状态。

### 1.3 解决的问题
- 现代创作者面临海量通知、社交软件、浏览器标签等干扰，难以持续专注
- 传统写作工具界面冰冷、字体机械，无法唤起书写的仪式感与创作热情
- 频繁在不同应用间切换，注意力被不断切割

### 1.4 核心价值
- **效率提升**：通过沉浸式环境设计，帮助用户缩短进入心流状态的时间，提升连续写作时长
- **仪式感**：让屏幕写作回归纸笔的温度，唤醒书写仪式感
- **专注体验**：屏蔽干扰，创造纯粹的写作空间

## 2. 目标用户

### 2.1 用户群体
| 用户类型 | 描述 | 使用场景 |
|---------|------|---------|
| 自媒体作者 | 内容创作者、博主 | 文章创作、公众号写作 |
| 学术研究者 | 学生、学者、研究员 | 论文写作、研究笔记 |
| 小说创作者 | 网络作家、小说家 | 长篇小说创作 |
| 技术写作者 | 程序员、技术专家 | 技术文档、博客撰写 |
| 自由撰稿人 | 自由职业写作者 | 各类稿件撰写 |
| 普通用户 | 日记爱好者、学生 | 日记、随笔记录 |

### 2.2 用户痛点
- 界面复杂，干扰元素多
- 字体机械冰冷，缺乏温度
- 无法有效屏蔽系统级干扰
- 难以进入并维持专注状态
- 缺乏写作仪式感

## 3. 功能需求

### 3.1 核心功能

#### 3.1.1 仿真稿纸界面
- 横线格稿纸背景
- 竖线格稿纸背景
- 纸张纹理效果
- 可调节行间距
- 可调节纸张尺寸

#### 3.1.2 手写书法字体渲染
- 楷体风格
- 行书风格
- 隶书风格
- 草书风格（可选）
- 字体大小调节

#### 3.1.3 纸张主题切换
- 米黄稿纸主题
- 仿古宣纸主题
- 纯白纸张主题
- 自定义主题颜色

#### 3.1.4 沉浸式写作模式
- 全屏写作模式
- 专注模式（隐藏干扰元素）
- 打字声音反馈（可选）
- 字数统计
- 写作时长追踪

#### 3.1.5 文档管理
- 本地存储（LocalStorage）
- 自动保存
- 文档导出（TXT/MD）
- 文档命名与管理

### 3.2 辅助功能

#### 3.2.1 快捷键支持
- Ctrl/Cmd + S：保存
- Ctrl/Cmd + Enter：换行
- Esc：退出全屏
- Ctrl/Cmd + B：加粗（概念模式）

#### 3.2.2 写作统计
- 实时字数统计
- 写作时长记录
- 日均写作统计
- 写作进度追踪

---

## 4. 非功能需求

### 4.1 性能需求
- 首屏加载时间 < 2s
- 打字响应延迟 < 100ms
- 支持离线使用

### 4.2 兼容性需求
- 支持 Chrome、Firefox、Safari、Edge 主流浏览器
- 支持桌面端（Windows、macOS、Linux）
- 支持平板端（iPad、Android 平板）
- 响应式设计适配不同屏幕尺寸

### 4.3 可用性需求
- 界面简洁，学习成本低
- 操作直观，无需教程即可上手
- 响应迅速，无卡顿感

### 4.4 安全性需求
- 数据本地存储，不上传服务器
- 支持数据导出备份
- 隐私保护，不收集用户数据

---

## 5. 技术要求

### 5.1 技术栈
- 前端框架：Vue 3 + TypeScript
- 构建工具：Vite
- CSS 框架：TailwindCSS 3
- 图标库：Lucide Icons
- 状态管理：Vue Composition API
- 本地存储：LocalStorage API

### 5.2 核心技术实现
- Canvas/SVG 渲染稿纸背景
- Web Fonts 加载手写字体
- 文本编辑区使用 contenteditable
- 响应式设计适配多端

### 5.3 开发环境
- Node.js >= 18.0.0
- npm >= 9.0.0

## 6. 用户故事

### 用户故事 1：进入沉浸式写作
**作为** 一名自媒体作者  
**我希望** 能够进入一个无干扰的写作环境  
**以便** 专注完成文章创作

**验收标准：**
- 点击全屏按钮后进入沉浸式模式
- 所有干扰元素（菜单栏、通知等）隐藏
- 仅显示写作区域和必要的工具栏

### 用户故事 2：切换纸张主题
**作为** 一名小说创作者  
**我希望** 能够切换不同的纸张主题  
**以便** 根据写作内容选择合适的书写氛围

**验收标准：**
- 提供至少 3 种预设纸张主题
- 点击主题切换按钮即可更换
- 主题切换后立即生效

### 用户故事 3：使用手写字体写作
**作为** 一名学术研究者  
**我希望** 能够使用手写风格的字体写作  
**以便** 获得更具温度的写作体验

**验收标准：**
- 支持至少 3 种书法风格（楷体、行书、隶书）
- 字体切换后即时生效
- 文字渲染清晰美观

### 用户故事 4：查看写作统计
**作为** 一名技术写作者  
**我希望** 能够查看我的写作统计数据  
**以便** 了解自己的创作节奏

**验收标准：**
- 显示实时字数统计
- 显示当前写作时长
- 显示历史写作记录

### 用户故事 5：保存和导出文档
**作为** 一名自由撰稿人  
**我希望** 能够保存和导出我的文档  
**以便** 备份和分享我的作品

**验收标准：**
- 自动保存文档到本地
- 支持导出为 TXT 格式
- 支持导出为 Markdown 格式
